About this book

Our Little Memories is your own handmade scrapbook. Without a sync server, everything you add stays only on this device, in this browser.

Sync server (optional)

Connect this book to a free online database (Firebase) so every device using the same link sees the same pages. Paste your database link below — see the guide for the 5-minute free setup.

Reset book

This permanently erases every page you've created on this device. There's no undo.
